### QR Code Content ###

* The QR code for now should contain a string that has the following format:
* ServerName|Key|IP|Port|Type
* ServerName: String
* Key: Encryption Key
* IP: XXX.XXX.X.XXX
* Port: 0 ... 65535
* Type: 0 = PC; 1 = Mac

* Example: MyNotebook|X4Eg5jKo0Xw4|192.168.1.69|6232|1
